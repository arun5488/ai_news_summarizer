# ai_news_summarizer


To run the app, following API Keys to be updated in .env file - 
OPENAI_API_KEY, 
TAVILY_API_KEY

App gets launched through app.py

Given a topic, this app will fetch the news articles from the web and summarize the topic's developments in the frequency provided in "Select Time Frame" dropdown.
For example - if the topic name is IPL 2025, app will return weekly summaries from the start to end of IPL 2025. 


the report also gets stored as a .MD file in the AINews Folder
